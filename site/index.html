Projekt nieoficjalny. Nie jest tworzony ani wspierany przez Fakturownia sp. z o.o. Korzysta z publicznego API Fakturowni i działa wyłącznie w trybie odczytu — nie wystawia, nie edytuje ani nie usuwa faktur.

Co to robi

Rozmowa zamiast klikania po panelu.

Serwer łączy się z Twoim kontem przez API i liczy odpowiedzi po swojej stronie — salda, przeterminowania, rankingi — więc wyniki są szybkie i spójne. Zapytaj o cokolwiek z tych:

„

Kto mi nie zapłacił i ile jest po terminie?

„

Jaki był obrót w czerwcu? Porównaj z majem.

„

Kto jest moim największym klientem w tym roku?

„

Rozbij sprzedaż VAT za II kwartał.

„

Instalacja · Claude Desktop

Gotowe w cztery kroki.

Nie musisz instalować Node.js — Claude Desktop dostarcza własny. Paczka jest niepodpisana, więc przy instalacji zobaczysz „nieznany wydawca” (normalne przy dystrybucji spoza oficjalnego katalogu).

Weź kod API

W Fakturowni: Ustawienia → Ustawienia konta → Integracja → Kod autoryzacyjny API.

Zapamiętaj domenę

Część przed .fakturownia.pl — dla mojafirma.fakturownia.pl to mojafirma.

Zainstaluj rozszerzenie

Pobierz .mcpb, a w Claude Desktop otwórz Ustawienia → Extensions → Install Extension… i wskaż plik. Na macOS wystarczy dwuklik.

Podaj dane i sprawdź

Wpisz domenę i token — token trafia do systemowego sejfu. Potem zapytaj:

„Pokaż obrót z tego miesiąca”
Narzędzia

Dwanaście narzędzi, jeden token.

Okresy (period): this_month, last_month, last_30_days, this_year, last_year, last_12_months lub własny zakres dat.

Bezpieczeństwo

Twoje dane zostają u Ciebie.

Tylko odczyt

Serwer wykonuje wyłącznie zapytania GET — nie wystawia, nie edytuje ani nie usuwa faktur.

Token w sejfie

Podajesz go przy instalacji; trafia do systemowego sejfu (Keychain / Credential Manager). Nigdy nie opuszcza oficjalnego API Fakturowni.

Kod jest jawny

Cały serwer to jeden plik bez zaciemniania. Paczkę .mcpb możesz rozpakować (to zwykły ZIP) i sprawdzić zawartość.

Bez natywnych binariów

Zależności to czysty JavaScript. Jeśli token wycieknie — wygeneruj nowy w ustawieniach Fakturowni.
