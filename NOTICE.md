# Oświadczenie o niezależności

**To jest projekt nieoficjalny.** Nie jest tworzony, wspierany, sponsorowany
ani w żaden sposób powiązany z firmą Fakturownia sp. z o.o. ani z serwisem
Fakturownia.pl.

- „Fakturownia" jest znakiem towarowym należącym do jego właściciela i użyto go
  tu wyłącznie w celu opisowym — dla wskazania, z jakim API ten serwer się łączy.
- Projekt korzysta z publicznie udokumentowanego API Fakturowni
  (https://github.com/fakturownia/API) na zasadach określonych przez dostawcę.
- Autor projektu nie ponosi odpowiedzialności za działanie API Fakturowni,
  zmiany w nim, ani za skutki użycia tego oprogramowania. Oprogramowanie jest
  dostarczane „tak jak jest" (patrz LICENSE).

Jeśli reprezentujesz Fakturownię i chcesz zgłosić zastrzeżenia dotyczące nazwy
lub znaku towarowego — otwórz issue w repozytorium.
